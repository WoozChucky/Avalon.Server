    Spell System¶

    This document describes the lifecycle of a spell from definition to client animation, covering all open design gaps.

    Related TODOs: TODO-015, TODO-016, TODO-017, TODO-018, TODO-019, TODO-020

    Spell Data Model¶

    SpellTemplate (persisted — Avalon.Domain.World)¶

    Stored in the World database. Loaded at server startup into StaticData.SpellTemplates.

    Field Type Description
    Id SpellId Primary key
    Name string Display name
    CastTime uint Cast duration in milliseconds; 0 = instant
    Cooldown uint Cooldown in milliseconds
    Cost uint Power cost (mana / energy / fury depending on class)
    SpellScript string Script class name used to resolve the SpellScript type
    Range SpellRange Targeting range category
    Effects SpellEffect Flags: Damage, Heal, AoE, DoT, HoT, etc.
    EffectValue uint Base effect magnitude (damage dealt, healing done, etc.)
    AllowedClasses List<CharacterClass> Which classes can learn this spell
    AnimationId uint Client animation ID to play on cast (TODO-017)

    SpellMetadata (runtime — Avalon.World.Public.Spells)¶

    In-memory representation loaded from SpellTemplate. Immutable (uses init properties).

    public class SpellMetadata
{
    public string Name { get; init; }
    public float CastTime { get; init; }  // seconds
    public float Cooldown { get; init; }  // seconds
    public uint Cost { get; init; }
    public string ScriptName { get; init; }
    public SpellRange Range { get; init; }
    public SpellEffect Effects { get; init; }
    public uint EffectValue { get; init; }
    public uint AnimationId { get; init; } // TODO-017
}

    Spell Lifecycle¶

    Player presses spell key
        │
        ▼
CCharacterAttackPacket / CCastSpellPacket
        │
        ▼
CharacterAttackHandler.Execute
  ├── AoE check (TODO-020): skip target if SpellEffect.AoE
  ├── Single-target: validate target exists and is in range
  └── chunk.QueueSpell(character, target, spell)
        │
        ▼
ChunkSpellSystem.QueueSpell
  ├── Power check (TODO-015): character.CurrentPower >= spell.Metadata.Cost?
  │   └── NO → return false (client notified out-of-power)
  ├── Deduct power (TODO-015): character.CurrentPower -= Cost
  └── Add SpellInstance to _spellQueue
        │
        ▼
ChunkSpellSystem.Update (per-tick)
  ├── Decrement CastTimeTimer
  ├── Check for cast interruption (movement)
  └── CastTimeTimer <= 0
        │
        ├── Resolve SpellScript type via IScriptManager
        ├── Instantiate SpellScript via DI
        └── SpellScript.Prepare() → add to _activeSpells
              │
              ▼
        SpellScript.Update (per-tick)
          └── Script logic: hit detection, projectile movement, DoT ticks, etc.
              │
              ▼
        On impact / completion
          ├── Apply damage / heal to target
          ├── Broadcast SUnitDamagePacket or SHealPacket
          └── BroadcastFinishCastAnimation (uses AnimationId — TODO-018)

    Power Cost Deduction (TODO-015)¶

    Power deduction happens at queue time (when the spell is accepted into the queue), not at cast completion. This prevents exploiting cast interruptions to "bank" the power refund.

    Power Types¶

    PowerType Class(es) Mechanic
    Mana Wizard, Healer Depleted on cast; regenerates over time
    Energy Hunter Depleted on cast; regenerates quickly
    Fury Warrior Accumulation mechanic (defers TODO-015)
    None — No resource check

    For PowerType.Fury, do not deduct on cast; Fury is generated by successful melee attacks and consumed differently. Defer this mechanic to a dedicated Fury design story.

    SpellScript.Clone() (TODO-016)¶

    SpellScript instances are cloned from a prototype when a spell is activated. The current abstract Clone() forces each script to implement its own clone.

    Proposed Base Implementation¶

    public virtual SpellScript Clone()
{
    // Shallow clone of all value-type fields
    var clone = (SpellScript)MemberwiseClone();

    // Deep-clone the chained scripts list
    var clonedChain = new List<SpellScript>(ChainedScripts.Count);
    foreach (var chained in ChainedScripts)
        clonedChain.Add(chained.Clone());

    // Assign via backing field (ChainedScripts is a protected List)
    clone._chainedScripts = clonedChain;

    return clone;
}

    Concrete scripts with additional mutable state (e.g. an elapsed timer field, a per-instance position offset) should override Clone() and call base.Clone(), then reset their mutable fields:

    public override SpellScript Clone()
{
    var clone = (FireballScript)base.Clone();
    clone._elapsedTime = 0f;
    return clone;
}

    CreatureCombatScript.AttackTarget Logic¶

    AttackTarget(deltaTime)
  ├── Check if any spell is off cooldown and in range
  │   ├── YES → QueueSpell(Creature, target, spell); start cooldown
  │   └── NO → melee attack (existing logic)
  └── (end)

    Cast restriction: A creature with a CastTime > 0 spell may not cast while moving (same rule as players, enforced by ChunkSpellSystem).

    AoE Targeting (TODO-020)¶

    AoE spells have SpellEffect.AoE in their Effects flags. They do not require a single target in the attack packet.

    Flow Change in CharacterAttackHandler¶

    // Resolve spell from character spell list using packet.SpellId
ISpell? spell = character.GetSpell(packet.SpellId);

if (spell?.Metadata.Effects.HasFlag(SpellEffect.AoE) == true)
{
    // AoE: no target required
    chunk.QueueSpell(character, null, spell);
    return;
}

// Non-AoE: require target (existing flow)
ObjectGuid targetGuid = new(packet.Target);
IUnit? target = GetTarget(chunk, targetGuid);
if (target == null) { ... return; }
chunk.QueueSpell(character, target, spell);

    AoE SpellScript Responsibilities¶

    The script must find affected targets within its radius during Update() / Prepare():

    protected override void Prepare()
{
    // Collect all units within Spell.Metadata.Range of Caster.Position
    var targets = Chunk.Creatures.Values
        .Where(c => Vector3.Distance(c.Position, Caster.Position) <= Range)
        .Cast<IUnit>();

    foreach (var t in targets)
        t.TakeDamage(Caster, Spell.Metadata.EffectValue);
}
